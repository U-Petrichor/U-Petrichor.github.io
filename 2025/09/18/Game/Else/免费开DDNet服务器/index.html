免费开DDNet服务器

发表于2025-09-18|更新于2025-09-24|游戏ElseDDNet
|总字数:1.9k|浏览量:|评论数:

前言

2025年9月初，DDNet国服似乎不运营了，大陆玩家只能去连港服，然而港服延迟较高，还非常不稳定，即使开了加速器也很糟糕，外服虽然稳定一些，但是延迟更高，所以说为了能玩到更加稳定的服务器，我折腾了一个下午找到了个免费开服的方法，根据本文你就可以实现开服与朋友联机，不过这个方法也是有一些缺陷，就是服务器不会显示在DDNet的互联网频道，所以说聚集在一起是很重要的（咳咳，Q群：970039533）

后续惊喜！！
我不知道什么原理，居然有路人可以直接加入到我的服务器！！！

本文是保姆式教程，电脑0基础的玩家跟着做也可以实现联机

了解开服的事项

这一个大标题如果你着急看开服可以不看，只是如果后续碰到一些问题或者自己钻研新方法的时候可能会对你有帮助

官网提供的信息

打开DDNet官网然后翻译之后可以看到主页面有own server的链接，点击即可跳转
跳转之后有许多选项，选择Server Setting就可以跳转到有关服务器设置的相关内容，这里你可以根据其解释来进行设置，所以请自己备好翻译

要配置的文件在哪

我们先尝试点击游戏里的运行服务器，发现弹出来一个终端，有这些内容，具体看图中解析


唉！到现在我们已知的内容有哪些？

  • 使用的是udp通信（这个其实蛮重要的，因为不是所有免费的内网穿透都支持udp，大部分都是支持tcp）
  • 端口是8303起
    所以到这里其实我有两个思路，一个是使用Radmin LAN开局域网，还有就是想办法使用内网穿透，如果说你有其他思路但是自己没有尝试的话，欢迎在评论区分享，我有空会尝试并且根据结果更新文档

开服方法

  • 目前只研究了一个，后续可能还会折腾一些新方法，所以现在这里了个二级标题

SakuraFrp 樱花内网穿透

视频版教程，但是肯定没文章详细

没想到想当年MC联机的方法还能用到DDNet上，内网穿透的原理就是将本地的端口映射到公网的端口，这样就可以通过公网的端口访问到本地的服务了，而SakuraFrp就是一个免费的内网穿透工具，它支持udp和tcp，所以说我们可以使用它来开服（还有个比较出名的内网穿透工具ngrok，这个并不支持udp传输，至少2025/9/18这天是不行的）

配置SakuraFrp

  1. 打开SakuraFrp的官网
  2. 点击注册，注册流程我就不说了
  3. 跳转到下一个页面之后，点击Sakura Frp
  4. 优先完成实名认证
  5. 按照图中的流程找到创建隧道
  6. 任选一个节点，根据服务器的状态自己选择，注意上面有支持 UDP的选项，我这里选择的是东莞双线PLUS2
  7. 按照如图所示创建隧道即可

下载客户端并启用

  1. 点击下载客户端，根据自己的操作系统下载对应的客户端，具体的下载流程请根据下载页的教程一步步执行，我这里就不提供了
  2. 先根据图中位置”复制密钥“
  3. 再把密钥复制到安装好的客户端，点击登录
  4. 登录之后会有如图所示的两个选项，选择有UDP的这一项选择开启
  5. 开启之后根据图找到需要复制的内容

准备并开始联机

  1. 先找到配置的文件，根据图找到autoexec_server.cfg文件

  2. 回头看看我一开始提到的有关服务器配置信息的网站，我这里只会做必须项的修改
  3. 打开autoexec_server.cfg文件（鼠标右击之后打开方式选择记事本或者VScode之类的都行）
  4. 按ctrl+f，搜索sv_register，将这个参数后面的数字改为1，这个为必须项，如果你想要服务器有一些额外的限制，请自行看上面说的官网
  5. 按ctrl+s保存文件，然后打开你的DDNet，点击运行服务器
  6. 终端会出现这个报错，但是不要在意，出现这个报错的原因是DDNet服务器想要连接连接你的IPv4地址的8303端口，但是因为我们使用的都是经过NAT转换的内网，所以说DDNet服务器是连不上的，而内网穿透就像是一个桥梁，别人先帮你接收，然后再转交给你，所以说只要别人把数据传到要转交给你的人，你一样可以实现数据传输
  7. 让朋友在互联网这里输入你刚刚在Sakura Frp里复制的内容，即可加入到你的服务器
  8. 加入成功的图片
  9. 后续意外之喜，路人也可以进入，但是暂时没搞清楚路人那里是什么原理

Radmin VPN联机

这也是曾经用来Minecraft联机时学到的方法，如今深入了解了一下，原理大致是将所有人连接到了一个虚拟的局域网中，所以理论上必须加入这个局域网才可以进行联机，所以这个方法你的地图可能不会在互联网列表中出现（我自己并没有尝试，希望后续如果有人使用此方法联机，有路人进入服务器当中，请告知我我会修改此处）

下载和安装Radmin VPN

  1. 打开Radmin VPN的官网，点击免费下载
  2. 下载了exe文件之后直接安装就好（没有任何需要设置的，我还没反应过来就安装完了）

加入局域网

  1. 根据图中所示找到创建网络
  2. 输入网络名称和密码
  3. 你的好友需要使用你刚刚填写的网络名称和密码加入到这个局域网中

  4. 加入之后服主打开游戏点击运行服务器，然后所有加入局域网的人都可以在局域网那一栏找到服务器
  5. 似乎这个延迟会低一点
